🔔 Bounce/Complaint Webhook Kurulumu Otomatik Suppression
Bounce ve şikayet bildirimlerini otomatik almak için servisinizin panelinde aşağıdaki URL'leri tanımlayın. Gelen bildirimler suppression listesine otomatik eklenir.
🟦 Brevo
Brevo Dashboard → Settings → Webhooks → Add a new webhook
Events: hard_bounce, spam, unsubscribe
🟩 Mailrelay
Mailrelay Panel → Configuración → Notificaciones → Añadir URL
Events: bounce, complaint, unsubscribe
🟧 AWS SES (SNS)
1) AWS SNS → Create Topic (Standard) → Topic ARN'ı kopyala
2) Topic → Create Subscription → Protocol: HTTPS → Endpoint: yukarıdaki URL
3) SES → Configuration Sets → Event Destinations → SNS → Bounce + Complaint seç
4) İlk POST geldiğinde abonelik otomatik onaylanır.
